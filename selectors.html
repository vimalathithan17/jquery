jQuery Selectors Example

Selector Usage
$("#id") Select element by id
$(".class") Select elements by class
$("element") Select elements by tag name (e.g., $("p") for all <p> tags)
$("*") Select all elements
$("parent child") Select descendants (any level deep)
$("parent > child") Select direct children
$("prev + next") Select next adjacent sibling
$("prev ~ next") Select all siblings after prev
$("[attribute]") Select elements with a specific attribute
$("[name='value']") Select elements where the attribute has a specific value
$("[attr^='val']") Attribute starts with value
$("[attr$='val']") Attribute ends with value
$("[attr*='val']") Attribute contains value
$(":input") Select all form input elements
$(":text") Select all text inputs
$(":checkbox") Select all checkboxes
$(":radio") Select all radio buttons
$(":checked") Select checked elements
$(":first") Select first element in a group
$(":last") Select last element in a group
$(":even") Select elements with even index (0-based)
$(":odd") Select elements with odd index
$(":visible") Select visible elements
$(":hidden") Select hidden elements
$(":contains('text')") Select elements that contain specific text

Function Syntax Description Parameters Optional? Default Value
.text() $(selector).text()
$(selector).text(value)		 Gets or sets the text content of elements. value (string or function) - The new text content. ✅ Returns current text if no parameter is passed.
.html() $(selector).html()
$(selector).html(value)		 Gets or sets the HTML content of elements. value (string or function) - The new HTML content. ✅ Returns current HTML if no parameter is passed.
.val() $(selector).val()
$(selector).val(value)		 Gets or sets the value of form elements. value (string or function) - The new input value. ✅ Returns current value if no parameter is passed.
.append() $(selector).append(content) Adds content to the end of selected elements. content (string, element, or jQuery object) - Content to append. ❌ -
.prepend() $(selector).prepend(content) Adds content to the beginning of selected elements. content (string, element, or jQuery object) - Content to prepend. ❌ -
.after() $(selector).after(content) Inserts content immediately after the selected elements. content - HTML, element, or jQuery object to insert. ❌ -
.before() $(selector).before(content) Inserts content immediately before the selected elements. content - HTML, element, or jQuery object to insert. ❌ -
.remove() $(selector).remove() Removes selected elements and their data. None ✅ -
.empty() $(selector).empty() Removes all child elements from the selected elements. None ✅ -
.detach() $(selector).detach() Removes elements but keeps data and events. None ✅ -
.wrap() $(selector).wrap(wrapper) Wraps selected elements inside the given wrapper. wrapper (string or element) - Wrapper element. ❌ -
.wrapAll() $(selector).wrapAll(wrapper) Wraps all selected elements inside a single wrapper. wrapper (string or element) - Wrapper element. ❌ -
.wrapInner() $(selector).wrapInner(wrapper) Wraps the inner content of selected elements. wrapper (string or element) - Wrapper element. ❌ -
.unwrap() $(selector).unwrap() Removes the parent of selected elements. None ✅ -
.attr() $(selector).attr(name, value) Gets or sets attributes of elements. name - Attribute name.
value - Attribute value. 		✅ Returns current value if no parameter is passed.
.removeAttr() $(selector).removeAttr(name) Removes specified attributes. name - Attribute name to remove. ❌ -
.prop() $(selector).prop(name, value) Gets or sets properties of elements. name - Property name.
value - New property value. 		✅ Returns current value if no parameter is passed.
.removeProp() $(selector).removeProp(name) Removes specified properties. name - Property name to remove. ❌ -

jQuery DOM Manipulation Example

Method Description
text("new text") Changes the text inside an element
html("<b>new html</b>") Replaces the content with new HTML
val("new value") Changes the value of an input field
append("new element") Adds content at the end of an element
prepend("new element") Adds content at the beginning of an element
after("new element") Inserts new content after an element
before("new element") Inserts new content before an element
appendTo(selector) Moves an element inside another element at the end
prependTo(selector) Moves an element inside another element at the beginning
remove() Removes an element and its events
empty() Clears all child elements inside a container
detach() Removes an element but keeps its events
wrap("<div>") Wraps an element inside another element
wrapAll("<div>") Wraps all selected elements inside one wrapper
unwrap() Removes the wrapper of an element
attr("attrName", "value") Sets an attribute for an element
removeAttr("attrName") Removes an attribute from an element

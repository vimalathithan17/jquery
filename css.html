jQuery CSS Manipulation Methods

Method Syntax Description
.css() $(selector).css(property, value) or $(selector).css({"property1":"value1","property2:value2"}) Gets or sets CSS properties for selected elements.
.addClass() $(selector).addClass(className) Adds one or more classes to the selected elements.
.removeClass() $(selector).removeClass(className) Removes one or more classes from selected elements.
.toggleClass() $(selector).toggleClass(className) Adds/removes a class (toggles) based on its current state.
.hasClass() $(selector).hasClass(className) Checks if any selected element has a specific class.
.height() $(selector).height() Gets or sets the height of selected elements (excludes padding & border).
.width() $(selector).width() Gets or sets the width of selected elements (excludes padding & border).
.innerHeight() $(selector).innerHeight() Gets the height including padding (excludes border & margin).
.innerWidth() $(selector).innerWidth() Gets the width including padding (excludes border & margin).
.outerHeight() $(selector).outerHeight(includeMargin) Gets the height including padding & border (optionally margin).
.outerWidth() $(selector).outerWidth(includeMargin) Gets the width including padding & border (optionally margin).
.offset() $(selector).offset() Gets or sets the coordinates relative to the document.
.position() $(selector).position() Gets the position relative to the nearest positioned ancestor.
.scrollTop() $(selector).scrollTop() Gets or sets the vertical scroll position of an element.
.scrollLeft() $(selector).scrollLeft() Gets or sets the horizontal scroll position of an element.

jQuery Effects Methods Example

Method Description
show(speed) Gradually shows the element
hide(speed) Gradually hides the element
toggle(speed) Toggles visibility
fadeIn(speed) Fades in the element
fadeOut(speed) Fades out the element
fadeToggle(speed) Toggles fade effect
fadeTo(speed, opacity) Fades to a specific opacity
slideDown(speed) Slides the element down
slideUp(speed) Slides the element up
slideToggle(speed) Toggles slide effect
animate({properties}, speed) Animates CSS properties
stop() Stops the current animation
delay(ms) Delays an animation

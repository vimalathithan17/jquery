jQuery DOM Traversal Methods

Method Syntax Description Parameters Return Value
.parent() $(selector).parent([filter]) Gets the direct parent of each element in the set. filter (optional) - A selector to filter the parent. jQuery object (parent elements)
.parents() $(selector).parents([filter]) Finds all ancestors (parents, grandparents, etc.). filter (optional) - A selector to filter ancestors. jQuery object (all ancestors)
.parentsUntil() $(selector).parentsUntil([stop], [filter]) Selects all parent elements up to (but not including) the specified stop element. stop (optional) - Selector to stop at.
filter (optional) - Filter matching parents. 		jQuery object (ancestors up to stop element)
.closest() $(selector).closest(selector) Finds the closest matching ancestor (including itself). selector - A selector to match. jQuery object (closest matching element)
.children() $(selector).children([filter]) Finds direct child elements. filter (optional) - A selector to filter children. jQuery object (direct children)
.find() $(selector).find(selector) Finds all descendants matching a selector. selector - A selector string. jQuery object (matching descendants)
.siblings() $(selector).siblings([filter]) Finds all siblings of an element. filter (optional) - A selector to filter siblings. jQuery object (siblings)
.next() $(selector).next([filter]) Finds the immediately following sibling. filter (optional) - A selector to filter the next sibling. jQuery object (next sibling)
.prev() $(selector).prev([filter]) Finds the immediately preceding sibling. filter (optional) - A selector to filter the previous sibling. jQuery object (previous sibling)
.nextAll() $(selector).nextAll([filter]) Finds all following siblings. filter (optional) - A selector to filter the siblings. jQuery object (all next siblings)
.prevAll() $(selector).prevAll([filter]) Finds all preceding siblings. filter (optional) - A selector to filter the siblings. jQuery object (all previous siblings)
.nextUntil() $(selector).nextUntil([stop], [filter]) Finds all following siblings up to the stop element. stop (optional) - Selector to stop at.
filter (optional) - Filter matching elements. 		jQuery object (all next siblings until stop element)
.prevUntil() $(selector).prevUntil([stop], [filter]) Finds all preceding siblings up to the stop element. stop (optional) - Selector to stop at.
filter (optional) - Filter matching elements. 		jQuery object (all previous siblings until stop element)
.first() $(selector).first() Finds the first element in a jQuery object. None jQuery object (first matched element)
.last() $(selector).last() Finds the last element in a jQuery object. None jQuery object (last matched element)
.eq() $(selector).eq(index) Selects an element at a specific index. index - Zero-based index of element to select. jQuery object (element at the index)
.filter() $(selector).filter(selector) Filters the matched elements based on a selector. selector - A selector or function(params:index,element)to filter elements. jQuery object (filtered elements)
.not() $(selector).not(selector) Removes elements that match the selector. selector - A selector or function to exclude elements. jQuery object (non-matching elements)
.has() $(selector).has(selector) Finds elements that contain the specified child elements. selector - A selector to find contained elements. jQuery object (elements that contain matches)

jQuery Utility Functions Demo

Function Syntax Description Parameters Callback Syntax (if applicable)
$.each() $.each(object, callback) Iterates over an array or object and executes a function for each element.
  • object - The array or object to iterate over.
  • callback - Function to execute for each element.
 function(index, value) { /* logic */ }
.each() $(selector).each(callback) Iterates over a jQuery collection and executes a function on each element.
  • callback - Function to execute on each element.
 function(index, element) { /* logic */ }
$.map() $.map(array, callback) Transforms each array element and returns a new array.
  • array - The array to modify.
  • callback - Function to transform elements.
 function(element, index) { return transformedValue; }
.map() $(selector).map(callback) Transforms each element in a jQuery collection and returns a new jQuery object.
  • callback - Function to transform elements.
 function(index, element) { return transformedValue; }
$.merge() $.merge(array1, array2) Merges two arrays into the first array.
  • array1 - The target array.
  • array2 - The array to be merged into array1.
 N/A
$.grep() $.grep(array, callback, invert?) Filters elements in an array based on a condition.
  • array - The array to filter.
  • callback - Function that determines inclusion.
  • invert (optional) - If true, reverses selection.
 function(element, index) { return condition; }
$.trim() (Deprecated) $.trim(str) Removes leading/trailing whitespace. Use String.trim() instead.
  • str - The string to trim.
 N/A
$.now() $.now() Returns the current timestamp (milliseconds since 1970). N/A N/A
$.extend() $.extend(target, obj1, obj2, ...) Merges properties of objects into the target object.
  • target - The object to extend.
  • obj1, obj2, ... - Objects to merge.
 N/A
$.Deferred() $.Deferred() Creates a Deferred object for asynchronous operations. N/A $.Deferred().done(function(result) { /* handle success */ });

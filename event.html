jQuery Event Methods (Updated)

Event Method Description Example
submit() Triggers when a form is submitted
change() Triggers when an input value changes
focus() Triggers when an input gains focus
blur() Triggers when an input loses focus
keydown() Triggers when a key is pressed(here console logged)
keyup() Triggers when a key is released(here console logged)
click() Triggers when an element is clicked
dblclick() Triggers when an element is double-clicked
contextmenu() Triggers on right-click
mouseenter() Triggers when mouse enters an element
Hover Me
mouseleave() Triggers when mouse leaves an element
Leave Me
hover(fn1, fn2) Triggers on hover-in and hover-out
Hover
resize() Triggers when the window is resized Resize the window!
scroll() Triggers when the page is scrolled Scroll down!(here console logged)
on(event, selector, function) Binds events dynamically (For future elements too) (Dynamically added buttons)
off(event, selector, function) Removes an event dynamically
one(event, function) Triggers an event only once
